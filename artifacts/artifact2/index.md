# Artifact 2 – Machine Learning vs Deep Learning

Indiana Wesleyan University

Workshop 2

## Overview

This artifact explores the similarities and differences between Machine Learning (ML) and Deep Learning (DL), highlighting their architectures, learning approaches, strengths, limitations, and real-world applications.

The goal of this assignment was to evaluate where each technology is most effective and communicate these concepts through practical examples.

## Learning Objectives

Through this assignment I learned to:

- Differentiate Machine Learning and Deep Learning
- Analyze real-world AI applications
- Select appropriate AI techniques for specific problems
- Present technical concepts clearly
